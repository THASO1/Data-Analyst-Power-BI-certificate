# Professional Certification: Data Analyst in Power BI

![Certificate](Data%20Analyst%20in%20Power%20BI.png)

## 📌 Track Overview
I have successfully completed the **Data Analyst in Power BI** professional track on **DataCamp**. This intensive **50-hour** program is designed to build a career-ready skill set, covering the entire data analysis workflow from data discovery to sharing insights.

## 🛠 Technical Skills & Applied Projects
During this certification, I mastered the following core competencies required for a Data Analyst role:

* **Data Preparation:** Expertly using **Power Query** to extract, transform, and load (ETL) data from various sources.
* **Data Modeling:** Designing and optimizing relational data models, including creating star schemas and managing table relationships.
* **Advanced Analytics with DAX:** Writing complex **Data Analysis Expressions (DAX)** to create calculated columns and measures for business-critical KPIs.
* **Data Visualization:** Designing high-impact, interactive dashboards using best practices in data storytelling and user experience (UX).
* **Business Intelligence:** Performing exploratory data analysis (EDA) to provide actionable insights for data-driven decision-making.
